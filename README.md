### Support

DarkMode is a iOS Application with Swift language. Supporting : iOS 13.x and above, device orientation only portrait.

### Context

I created this app for the purpose of testing how to work with user mode switching (dark | light mode).

## Features

* Changement of UI according to the choice of user
  * White mode
  * Black mode

## Installation

First step:

```bash
Fork the project on your Github to have repo
```

Next step:

```bash
Git clone on your computer/laptop
```
